***** Generic is great thing *****

Generics write one version of a function or component that adapts to many types. It enforces strict type rules. Both flexibility and safety is obtained. There is no more sacrificing correctness for reusability.


Sometimes I got into problem for this generic declaring format. Specially when I tried to deal with object parameter, it gets worse. Whatever, somehow I managed to follow it's rule.


There is two main advantage:

1. Abstract type: It allows any types data without annotating at variable declaration.
2. DRY approach: It shorten very much lines of code into words in a resuable approach. That's amazing I must say.
